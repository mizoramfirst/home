MIZORAM FIRST

I ram leh hnam a boral chuan nang leh I chhungte in him bik lo vang.

INTRODUCTORY NOTE


MIZORAM FIRST pawl hi political party awn lam nei lo, Mizoram veitu kal khawmten March 2, 2022-a an din a ni. ‘Mizoram First’ tih chu ‘Mizoram dah pawimawh hmasak berna rilru’ tihna a ni ber a. Mimal hlawkna, party bil leh party member-te hlawkna aia Mizoram leh mipui thatna tur ngaihpawimawh hmasak duhna rilru, mikhual rilru ni loa neitu rilru putna hi ‘Mizoram First’ tih in a kawh ber chu a ni. 

Mizoram hi kan ram, kan ta a ni. A neitu kan nih avangin kan ram chhiat-ṭhat, a dinchhuahna leh boralna pawh Mizote kutah a awm. Kan chhehvelah keini aia tam leh fing zawkte, kan sakhua leh culture nena inang reng reng loten min hual vel a. Kan ram min neihsak chak em emte kan chenpui reng bawk. Chutih lai chuan Mizoram neitute chuan hnam hmangaihna thinlung tak tak pu lova, hmathlir engmah nei lova mahni hlawkna chauh kan um zel chuan kan ram leh hnam a boral dawn. 

Chuvangin, tuna Mizoramin a mamawh ber chu ram leh hnam hmangaihna (patriotism), mahni hlawkna tur aia ram leh hnam damna tur dah hmasak, Kristianna hming tibawrhbangtu ni lova chawisang tura rinawmna, dikna leh huaisenna nena ke pen a ni. Chu chu ‘MIZORAM FIRST’ in a tum ril ber a ni.

‘MIZORAM FIRST’ rilru put hi midang tan ni lovin, keimahni tan ṭheuh, kan chhungkua, kan veng, khawtlang leh hnam tan a tul a ni. He ram lo hi ram dang kan nei lo a, kan nei dawn hek lo. Ram leh hnam hmangaihna (patriotism) nunpuia, kan hnam leh kan thlahte tana chenna tlak ram siam hi mi tin mawhphurhna a ni tih mipui hriat thar tir hi ‘MIZORAM FIRST’ awm chhan a ni. 


Mission and Vision:

Mi tha leh dik, ram hmangaihtu an lo chhuah theihna tura kawng sial leh a chhe lai siam that. 


Thil tum (Objectives):

  1. Kan ram leh hnam hmangaihna rilru dik tak (Mizoram dah hmasak berna rilru) pu tura infuih thar.

  2. Mizo ngaihhlut thiltha (values) vawn nun leh chawi lar thar leh.

  3. Sorkar tha siam thei tura mipui buatsaih leh zirtir.

  4. Thalaite ram siamthatna hna thawk tura cho chhuah leh thawhpui.

  5. Kan ram leh hnam hmakhua atana ngaihtuahna senga thil tha tur chhawpchhuah dan ngaihtuah. 

  6. Ram hmangaihtu leh veitu zawng zawng an tel theihna tur hmun (platform) siam.

  7. Mitinin kan chanvo dik tak kan chan theih nana dan rorelna (‘rule of law’ leh ‘principle of justice’) kan rama a lal ber zel theih nana hma lak.

  8. Eirukna leh ram leh hnam him lohna tur thil dodal.


MIZORAM FIRST pawl hi political party-a inleh kan tum lo va. Ram vei tupawh lo tel turin kan insawm e.
